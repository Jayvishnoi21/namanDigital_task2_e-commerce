# E-Commerce Website

A Ecommerce Website made with React.js Framework.

## Features

- Easy to integrate with Backend
- Fully Responsive



## Run Locall

```bash
  cd React_E-Commerce
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
